# Guide for this section

This section provides information on the following:

1) **Applying methods on Testbed Models (TMs)**
The **CF** folder inside this contains code/data for applying CausalFlux on the testbed models. Look at the "README.md" file in this folder for more details
2) **Simulations** (To simulate ground truth for the testbed models): 
This section provides codes/data for generating the steady-state fluxes, and gene expression data for the TMs modelled through a system of ODEs. Look at the "README.md" file in this folder for more details
